> 版本：v1.0  
> 更新日期：2026-06-05

---

## 一、项目概述

### 1.1 项目背景

当前家长对儿童早期教育的需求日益增长，但市面上的学习工具要么功能单一（仅涂色/仅练字），要么依赖线下购买练习册，缺乏灵活性和个性化。童学绘旨在将"涂色、书写、练习、打印"整合到一个微信小程序中，让家长随时按需生成学习材料，孩子也能在手机上互动学习。

### 1.2 产品定位

面向 3-12 岁儿童的趣味学习工具，核心价值：

- **一键生成**：根据孩子年级和能力，智能生成练习纸
- **互动涂色**：手机上即可涂色，完成后保存/分享
- **打印可用**：所有练习纸均支持保存为图片，打印后纸笔使用
- **AI 加持**：错题同类生成、智能出题、分步骤讲解

### 1.3 目标用户

| 角色 | 描述 | 核心诉求 |
|------|------|---------|
| 家长 | 3-12岁儿童的家长 | 快速生成高质量练习纸，省去买练习册 |
| 儿童 | 3-12岁学前及小学阶段 | 涂色好玩、练习有趣、有成就感反馈 |

### 1.4 目标平台

- **主要**：微信小程序
- **导出方式**：Canvas 生成图片 → 保存到手机相册 → 打印使用

### 1.5 技术栈

| 层级 | 技术选型 |
|------|---------|
| 客户端 | UniApp (Vue 3 + TypeScript) + Wot UI + UnoCSS + Pinia |
| 服务端 | Koa + @koa/router + mysql2 |
| 数据库 | MySQL 8.0 |
| AI 服务 | DeepSeek / 通义千问等国内大模型 API |
| 文件存储 | OSS（线稿/笔画/素材资源） |

### 3.1 涂色绘画模块

#### F01 涂色绘本

| 项目 | 内容 |
|------|------|
| 功能描述 | 右上角展示已上色参考图，孩子在空白区域自由涂色 |
| 用户故事 | 作为家长，我想给孩子选一个涂色模板，让孩子参考示例图完成涂色，锻炼色彩认知和动手能力 |
| 输入 | 选择素材（动物/植物/交通工具等分类） |
| 输出 | 右上角小尺寸参考图 + 主区域空白线稿 + 底部调色板 |
| 交互流程 | 1.浏览素材列表 → 2.选择一个素材 → 3.右上角展示已上色参考图，主区域展示空白线稿 → 4.选色+点击/滑动涂色 → 5.完成后可保存图片 |
| 验收标准 | ① 参考图清晰可辨 ② 涂色区域边界准确，不溢出 ③ 支持12色以上调色板 ④ 支持撤销/橡皮擦 ⑤ 完成后可保存到相册 |
| 复杂度 | 高 |
| 依赖 | Canvas涂色引擎、SVG线稿素材库 |

#### F02 素描涂色

| 项目 | 内容 |
|------|------|
| 功能描述 | 生成黑白线稿，孩子自由选色涂色 |
| 用户故事 | 作为儿童，我想选一张喜欢的线稿涂上我喜欢的颜色，自由创作 |
| 输入 | 选择素材分类 |
| 输出 | 全屏空白线稿 + 底部调色板 |
| 交互流程 | 1.浏览线稿列表 → 2.选择线稿 → 3.自由涂色 → 4.保存作品 |
| 验收标准 | ① 线稿清晰无锯齿 ② 自由画笔+区域填充两种模式 ③ 支持保存作品 |
| 复杂度 | 中 |
| 依赖 | Canvas涂色引擎、SVG线稿素材库 |

#### F10 对称画/镜像涂色

| 项目 | 内容 |
|------|------|
| 功能描述 | 给出半张图案（蝴蝶/花瓶/人脸等），孩子画出另一半并涂色，训练观察与对称思维 |
| 用户故事 | 作为家长，我想让孩子通过画对称图案训练空间观察力和对称思维 |
| 输入 | 选择对称画素材 |
| 输出 | 左半已绘制+右半空白 + 中轴虚线 + 调色板+画笔 |
| 交互流程 | 1.选择对称画素材 → 2.显示左半已完成图案，中轴线标注 → 3.孩子在右半区域自由描绘并涂色 → 4.完成后可对比查看完整对称效果 |
| 验收标准 | ① 中轴线清晰标注 ② 左半图案固定不可修改 ③ 右半支持自由画笔+涂色 ④ 完成后可生成完整对称效果图 |
| 复杂度 | 高 |
| 依赖 | Canvas涂色引擎、对称画素材库 |

#### F11 按数字/字母涂色

| 项目 | 内容 |
|------|------|
| 功能描述 | 类似"像素画"，每个区域标注数字或字母，孩子按对应颜色涂色，同时巩固字母数字认知 |
| 用户故事 | 作为家长，我想让孩子在涂色的同时学习数字和字母的对应关系 |
| 输入 | 选择像素画素材 + 数字模式/字母模式 |
| 输出 | 带编号分区的线稿 + 颜色-编号对照表 + 调色板 |
| 交互流程 | 1.选择素材和模式 → 2.显示分区线稿，每区标注编号 → 3.底部显示"1=红, 2=蓝..."对照表 → 4.按编号选色涂色 → 5.完成后展示完整彩色图 |
| 验收标准 | ① 分区边界清晰 ② 编号字体大小适合儿童辨认 ③ 颜色对照表醒目 ④ 涂错颜色时提示 |
| 复杂度 | 高 |
| 依赖 | Canvas涂色引擎、像素画素材库（含分区和编号数据） |

#### F12 情绪涂色页

| 项目 | 内容 |
|------|------|
| 功能描述 | 给出不同表情的空白轮廓（开心/生气/害怕...），孩子涂色同时学习情绪表达与识别 |
| 用户故事 | 作为家长，我想让孩子通过涂色认识不同情绪表情，学习情绪表达 |
| 输入 | 选择情绪类型 |
| 输出 | 表情空白轮廓 + 情绪名称文字 + 调色板 |
| 交互流程 | 1.选择情绪表情 → 2.展示空白轮廓和情绪名称 → 3.自由涂色 → 4.涂色完成后展示情绪介绍小卡片 |
| 验收标准 | ① 至少6种基本情绪 ② 每种情绪有清晰的轮廓特征 ③ 涂色完成展示情绪介绍 |
| 复杂度 | 中 |
| 依赖 | Canvas涂色引擎、情绪素材库 |

#### F13 鼓励卡片生成

| 项目 | 内容 |
|------|------|
| 功能描述 | 涂色完成后自动生成"小画家证书"或"作品墙"图片，增强成就感 |
| 用户故事 | 作为儿童，我想在完成涂色后得到一张奖励证书，让我更有成就感 |
| 输入 | 已完成的涂色作品 |
| 输出 | 带作品缩略图的鼓励卡片图片（"小画家证书"/"作品墙"模板） |
| 交互流程 | 1.完成涂色 → 2.弹出"生成鼓励卡片"按钮 → 3.选择卡片模板 → 4.自动合成作品+装饰框+鼓励文字 → 5.保存卡片到相册 |
| 验收标准 | ① 至少3种卡片模板 ② 作品图清晰嵌入卡片 ③ 鼓励文字随机且积极 ④ 可保存到相册 |
| 复杂度 | 中 |
| 依赖 | Canvas图片合成、卡片模板 |
